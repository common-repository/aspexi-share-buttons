Simple social share (Facebook and Twitter) buttons for mobile, stick to the bottom of the page.

== Description ==

####About the plugin####

This awesome plugin lets your visitors share your content with pretty and nonintrusive buttons stuck to bottom of the screen.

Buttons are displayed on small screens only as they are designed for mobile users.

Two buttons are available: Facebook and Twitter share. You can enable both or one of them only.

Plugin if enabled is visible on all type of pages including homepage on small screens / mobile. You can test it on PC by reducing the size of browser window.

== Installation ==

1. Upload `aspexi-share-buttons` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin (Appearance -> Aspexi Share Buttons)
